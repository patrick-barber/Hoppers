What we have hear is a program that will take in the board locations of the pegs in the classic game known as Hoppers. It will determine what moves you have to make on the board in order to win.
------------------------------------------------------------------------

PROJECT TITLE: Hoppers
PURPOSE OF PROJECT: In order to practice my skills.
VERSION or DATE: Fall 2018
HOW TO START THIS PROJECT: Simply run the main method.
AUTHORS: Patrick Barber
USER INSTRUCTIONS: In order to change the board positions, open the Driver.java class and edit the existing board locations and compile and run the program again.
